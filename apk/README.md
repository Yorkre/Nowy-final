# APK de Nowy (debug)

Este directorio incluye el instalador Android:

- `app-debug.apk`: APK de prueba para instalar y ejecutar Nowy en un dispositivo Android.

## Descargar el APK

1. Haz clic en `app-debug.apk`.
2. Pulsa **Download raw file**.

## Instalar en Android

1. Copia el APK al teléfono.
2. Permite la instalación de apps de origen desconocido para la app desde la que lo abras.
3. Abre el archivo `app-debug.apk` y confirma la instalación.
